# AI 产品经理的 Agent 实战课 🚀

> 以 Claude Code 为教材，从「会用 AI」进阶到「会造 AI Agent 产品」

## 📖 在线阅读

**👉 [点击这里在线阅读教程](https://icapk.github.io/Claude-AIPM/)**

## 这是什么？

这是一份专为 **AI 产品经理** 设计的 Agent 学习教程。以 [claude-code-from-scratch](https://github.com/Windy3f3f3f3f/claude-code-from-scratch)（~1300 行 TypeScript）为教材，配合 Claude Code 官方源码（~51万行）作为进阶参照。

## 四个段位

| 段位 | 对 AI 的理解 |
|---|---|
| 🥉 L1 套壳 PM | "调 API 得结果" |
| 🥈 L2 特性 PM | "检索+生成，能调参" |
| 🥇 L3 架构 PM | "循环执行，工具驱动" |
| 💎 L4 引擎 PM | "理解引擎，能改规则验证想法" |

## 教程目录

| 章节 | 内容 | 段位要求 |
|---|---|---|
| 第 0 章 | 段位与能力矩阵 | 所有人 |
| 第 1 章 | 先跑起来再说 | 所有人 |
| 第 2 章 | Agent 循环——产品自主性的根基 | L2+ |
| 第 3 章 | 工具系统——能力边界定义 | L3+ |
| 第 4 章 | 提示词工程——产品人格与动态上下文 | L3+ |
| 第 5 章 | 上下文与成本——定价模型修正 | L3+ |
| 第 6 章 | 安全与信任设计 | L3+ |
| 第 7 章 | 从原型到生产——差距全景 | L4 |
| 第 8 章 | 实战武器库——Checklist + 练习 | L4 |

## 学习节奏

**5 天 × 2 小时 = 10 小时，从任何段位升级到 L4。**
